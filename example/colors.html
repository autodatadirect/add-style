Color Swatches

The signature color palette of Auto Data Direct, Inc. is Black and ADD Blue which creates a powerful differentiator for our brand. Using this palette appropriately and consistently creates an additional layer of distinction.

All swatches contain an example of the WCAG 2.0 color accessibility requirements that outline the legibility of an object based on its foreground and background color contrast. A baseline accepted contrast ratio is 2. WCAG requires a 3 for large elements and 4.5 or greator for small text. (Note: click the WCAG under each swatch to view the legibility chart. The text color in the swatch box is set to the best color contrast value. Higher the value the better.)

Our signature colors may be used extensively both for large areas of color and as accent colors.
