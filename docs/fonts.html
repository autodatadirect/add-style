    Headings and paragraphs

    All heading elements—e.g., <h1>—and <p> are reset to have their margin-top removed. Headings have margin-bottom: .5rem added and paragraphs margin-bottom: 1rem for easy spacing. To add an underline under the header add the class .underlineto it. Additionally, you can change its color and weight to default by adding classes: .black, .white or .bold.

    Heading Underline and Color

    Add .underline and .blue-600 class to heading to get the default underline and custom blue hr color (see Text Color section of the Color Palette for an usage example).

    Heading Underline Color

    Add .underline-black class to heading to get a black underline and header color. If you want a bold header header use the BS class .font-weight-bold.
